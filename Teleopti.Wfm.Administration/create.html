Create new Tenant

Lets you create new Tenant. You need to supply a user that have permissions to create database in the SQL Server.

You should also provide a user and a password that will be used connecting to the databases by the application. This should be new user/login that does not exist in the SQL Server.

Then you input the first Business Unit and the first Person that can logon and start configuring the new Tenant.
